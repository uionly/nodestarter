# telusdemo

#Add Item

Type : POST 
URL : localhost:5000/item

Request (type JSON) :
{"name":"Samsung","desc":"Is that a phone"}

Response :
{
    "desc": "Is that a phone",
    "_id": "5bbdff0b3d91e93a6c459e13",
    "name": "Samsung",
    "__v": 0
}
